# MOEX Quotes
A simple Python script for obtaining quotation lists of the Moscow Exchange and the quotes themselves.
## Dependencies:
Internet connection and `pandas`.
## How to use:
Just run it in the terminal like 
```sh
python3 script.py
```
and open the file `quotes.md ` in any text or Markdown editor. Column "LL" means "Listing Level".
